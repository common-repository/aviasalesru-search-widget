== Description ==

**Плагин больше не поддерживается и не обновляется**

**Пожалуйста, установите наш более функциональный плагин из каталога WP [Travelpayouts](https://wordpress.org/plugins/travelpayouts/)**

Подключайтесь к [Travelpayouts Travel Affiliate Program](https://www.travelpayouts.com/?utm_source=wp_org&utm_medium=aviasales_plugin).

Вы давно хотели монетизировать ваш блог на WordPress? Теперь нет ничего проще! Установите плагин Aviasales.ru для поиска билетов на вашем блоге и зарабатывайте на поиске дешёвых авиабилетов.

== Installation ==

1. Зарегистрируйтесь в партнёрской программе Aviasales.ru Travelpayouts, если вы этого ещё не сделали/
2. Скачайте плагин, разархивируйте его и скопируйте папку "aviasalesru-search-widget" в директорию "/wp-content/plugins/" на сервере, где расположен ваш блог.
3. Активируйте плагин в WordPress → Плагины.
4. Перенесите виджет в "Основную область виджетов" через меню WordPress → Внешний вид → Виджеты.
5. Настройте виджет: укажите свой ID партнёра, при необходимости измените цвета виджета, укажите заголовок.

Поздравляем! Виджет готов к работе. Стимулируйте пользователей искать авиабилеты на вашем блоге и зарабатывайте вместе с AviaSales.ru!
